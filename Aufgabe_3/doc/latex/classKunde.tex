\doxysubsection{Ausführliche Beschreibung}
Repraesentiert einen Kunden mit Einkaufsfunktionen. 

Die Klasse \doxylink{classKunde}{Kunde} stellt einen Kunden dar, der Einkaufsaktionen in einem \doxylink{classLager}{Lager} durchfuehren kann. Ein \doxylink{classKunde}{Kunde} hat einen Namen, eine Liste von Regalen, die er durchsuchen kann, und einen Warenkorb, um \doxylink{classArtikel}{Artikel} hinzuzufuegen.
